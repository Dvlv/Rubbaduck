# Rubbaduck

A simple chat bot used for 'Rubber Duck Debugging'

(see https://en.wikipedia.org/wiki/Rubber_duck_debugging)

## Why?

Fun. And to play with tkinter a bit.

## Dependencies

- Python 3
- tkinter

## Running

`python3 rubbaduck.py`

Then explain your code to the duck.
